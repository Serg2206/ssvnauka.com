## Почему мы это делаем

Научное знание — самое ценное, что создало человечество. Но современная система научных публикаций сломана:

- **Статьи теряются:** 90% научных работ никогда не цитируются
- **Доступ закрыт:** Paywall за $30-50 за одну статью
- **Формат устарел:** PDF создан для печати, не для экранов
- **Нет контекста:** Сложно понять, о чем статья, не прочитав 20 страниц
