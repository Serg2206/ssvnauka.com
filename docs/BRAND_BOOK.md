
# 🎨 Бренд-бук ssvnauka.com

## 1. Философия: "Digital Strict"
Сочетание академической строгости и цифровой динамики. Стиль: Cyberpunk Science / Future Tech.

## 2. Цветовая палитра (Neon Noir)
Проект использует только **Dark Mode**.

| Роль | Цвет | HEX | Описание |
| :--- | :--- | :--- | :--- |
| **Background** | Void Black | `#050505` | Глубокий черный, фон (не #000). |
| **Primary Text** | Paper White | `#EAEAEA` | Основной текст. |
| **Accent 1** | **Electric Blue** | `#00F0FF` | Структура, данные, IT, интеллект. |
| **Accent 2** | **Acid Lime** | `#CCFF00` | Действие (CTA), энергия, фокус. |
| **UI** | Graphite | `#1C1C1C` | Карточки, плашки. |

## 3. Типографика
* **Заголовки:** `Manrope` или `Unbounded` (Геометричные гротески).
* **Текст:** `Inter` (Читабельность).
* **Код/Данные:** `JetBrains Mono` (Технический стиль).

## 4. Логотип
* **Концепция:** "Двойной Слэш" `//`.
* **Текст:** `// ssvnauka`
* **Стиль:** Строгий белый шрифт, слэши — градиент от Blue к Lime.

## 5. UI Элементы
* **Кнопки:** Ghost Buttons (контурные) с неоновой обводкой и эффектом Glow при наведении.
* **Карточки:** Bento Grid. Темные блоки с микро-подсветкой границ.
* **Стекло:** Использование `backdrop-filter: blur` для хедера.

## 6. Анимации и эффекты
* **Hover States:** Плавные переходы с изменением цвета границ и добавлением неонового свечения.
* **Scroll Animations:** Использование Framer Motion для fade-in и slide-up эффектов при прокрутке.
* **Loading States:** Skeleton screens с градиентной анимацией.

## 7. Принципы дизайна
1. **Минимализм:** Убираем всё лишнее, оставляем только важное.
2. **Контраст:** Высокий контраст текста и фона для читабельности.
3. **Иерархия:** Четкое разделение уровней информации через размер, цвет и положение.
4. **Технологичность:** Использование монопространственных шрифтов и grid-систем.
5. **Респонсивность:** Адаптивный дизайн для всех устройств.

## 8. Tone of Voice
* **Уверенный:** Мы знаем, что делаем.
* **Прямой:** Без воды и общих фраз.
* **Технологичный:** Используем терминологию разработчиков и ученых.
* **Дерзкий:** Не боимся быть провокационными.

**Пример:** "Мы взломали код знания" вместо "Мы улучшаем доступ к научной информации".

## 9. Визуальные референсы
* **Вдохновение:** Cyberpunk 2077, GitHub Dark Theme, Arc Browser
* **Избегаем:** Академические сайты с белым фоном, устаревшие библиотечные системы
